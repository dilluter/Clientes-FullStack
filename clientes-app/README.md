# ClientesApp

Este projeto foi gerado com o Angular CLI versão 9.1.1.

---

## Estilização

Este projeto utiliza **Bootstrap 4** e o template **SB Admin**. A estilização é feita principalmente com classes prontas:

```html
<!-- Botão -->
<button class="btn btn-primary">Salvar</button>
 
<!-- Card -->
<div class="card">
  <div class="card-header">Título</div>
  <div class="card-body">Conteúdo</div>
</div>
```

Caso necessário, estilos personalizados podem ser adicionados em:

```
src/styles.css
```
---

##  Pré-requisitos

Antes de iniciar o projeto, instale:

- Node.js versão **16.20.2**
- Angular CLI versão **9.1.1**

Verifique a versão do Node:
```bash
node -v
```

---

##  Instalação de dependências

Acesse a pasta do projeto e instale as dependências:

```bash
npm install
```

---

##  Rodando o projeto

Execute:

```bash
npm run start
```

Acesse no navegador:

http://localhost:4200/

A aplicação será recarregada automaticamente ao salvar alterações.

---

##  Gerar código

Criar um novo componente:

```bash
ng generate component nome-do-componente
```

Outras opções:

```bash
ng generate directive|pipe|service|class|guard|interface|enum|module
```

---

##  Testes

Executar testes unitários:

```bash
ng test
```
