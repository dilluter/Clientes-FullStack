# ClientesApp

Este projeto foi gerado com o Angular CLI versão 9.1.1.

---

##  Pré-requisitos

Antes de iniciar o projeto, instale:

- Node.js versão **16.20.2**
- Angular CLI versão **9.1.1**

Verifique a versão do Node:
```bash
node -v
```

---

##  Instalação de dependências

Acesse a pasta do projeto Instale as dependências:

```bash
npm install
```

---

##  Rodando o projeto

Execute:

```bash
npm run start
```

Acesse no navegador:

http://localhost:4200/

A aplicação será recarregada automaticamente ao salvar alterações.

---

##  Gerar código

Criar um novo componente:

```bash
ng generate component nome-do-componente
```

Outras opções:

```bash
ng generate directive|pipe|service|class|guard|interface|enum|module
```

---

## 🧪 Testes

Executar testes unitários:

```bash
ng test
```
